Div with a css background-image sample

For it to work you need to use an image editing software like photoshop or the gimp and apply a blur to the image (e.g. gaussian blur). The more blurred the more steps of blur you can have when applying the blurIt effect. Save the image with a suffix. default: _blur

So if you have original.jpg you save the blurred one as original_blur.jpg

You can use it by calling $(element).blurIt(options); e.g.$("#original_image").blurIt({"blurIt":67}); This will apply a 67% of the max of the blurred image.

It works both with a background-image as with an img tag image.
